# Daniel_Mods - Chat-Helper
🔧 Чат‑хелпер для модераторов

Мини‑инструмент, который облегчает жизнь модерам. Он автоматически подсвечивает нужные слова и фразы красным, чтобы модератор мог быстро заметить нарушителя и выдать наказание без долгих поисков.

Что даёт:
- Быстрое обнаружение нарушителей
- Меньше ручной проверки
- Удобное визуальное выделение проблемных сообщений

Кастомизировать белый лист вы можете по пути: Daniel_Mods\Chat_Helper\white_list.txt.
Или же нажатием F7 в игре

Просто включил — и модерация стала в разы проще.
